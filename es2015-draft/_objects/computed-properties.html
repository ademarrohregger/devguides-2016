Computed properties

You can write properties with key names derived from expressions.

let id = 'john'
let Users = { [id]: 'John Frobisher' }
Users.john //=> 'John Frobisher'

You would've written it like the long way in ES5:

var id = 'john'
var Users = {}
Users[id] = 'John Frobisher'
