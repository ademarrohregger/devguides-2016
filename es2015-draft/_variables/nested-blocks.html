Nested blocks

Variables defined using let and const are available only inside {...} blocks, and other blocks inside them.

function run (user) {
  if (user) {
    let running = user.state === 'running'
    if (running) {
      `running` is still available here
    } else {
      and here, too
    }
  } else {
    but not here!
  }
}
